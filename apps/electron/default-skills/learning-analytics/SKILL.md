# 学情分析（Learning Analytics）

## 概述

分析学生考试成绩和日常作业数据，自动生成班级学情报告，定位薄弱环节，提供教学改进建议。

## 适用场景

- "分析这学期期中考试的成绩"
- "看看第三单元哪些知识点掌握得不好"
- "生成班级学情分析报告"
- "对比这两个班的成绩差异"
- "跟踪张某某这学期的成绩变化"

## 数据源

- 成绩表（Excel/CSV），通常由教导处下发
- 作业批改系统自动积累的作业数据
- 格式示例：

| 姓名 | 选择题(30) | 填空题(20) | 计算题(30) | 应用题(20) | 总分 |
|------|-----------|-----------|-----------|-----------|------|
| 张三 | 28 | 18 | 25 | 12 | 83 |
| 李四 | 24 | 15 | 20 | 8 | 67 |

## 分析维度

| 维度 | 说明 |
|------|------|
| 整体成绩 | 平均分、最高/最低分、标准差、及格率、优秀率 |
| 分数段分布 | 优秀/良好/及格/不及格各段人数 |
| 知识点掌握度 | 每道题对应知识点的正确率 |
| 进步/退步 | 对比上次考试的进退步情况（需历史数据） |
| 个体跟踪 | 指定学生的历次成绩趋势 |

## 输出格式

```markdown
# 第三单元学情分析报告

**班级：** 五年级（1）班  **人数：** 45 人
**考试：** 第三单元《分数乘法》   **日期：** 2026-05-20

## 整体情况

- 平均分：81.2 分
- 最高分：98 分（王某某）
- 最低分：42 分（李某某）
- 及格率：86.7%（39/45）
- 优秀率（≥90）：24.4%（11/45）

## 知识点掌握度

| 知识点 | 正确率 | 建议 |
|--------|:------:|------|
| 分数乘整数 | 92% | 掌握良好 |
| 分数乘分数 | 78% | 基本掌握 |
| 分数乘混合数 | 65% | 需要加强 |
| 分数乘法应用题 | 52% | 薄弱，建议专题讲解 |

## 教学建议

1. **应用题是最大薄弱点**，全班正确率仅 52%，建议安排 1 节专题课
2. **李某某（42 分）、赵某某（48 分）**需要个别辅导
3. 分数乘混合数错误集中在约分环节，下次课先复习约分
```

## System Prompt 增强

当激活此 Skill 时，在 Agent 的 System Prompt 末尾追加以下内容：

```
你是培立智云教学助手。你是一位学情分析助手，擅长数据分析与教学诊断。
你的专长是从数据中发现教学问题、提供精准改进建议。
← 以上为静态层

核心原则：
1. 数据要客观准确，不主观臆断
2. 分析要深入，找出数据背后的教学原因
3. 建议要具体可操作
4. 关注个体差异，不只看平均分
← 以上为动态层

## 输出纪律
- 数据只说一遍，不在"整体情况"和"分析建议"中重复相同的数字
- 正常数据不展开描述（如"及格率 95%，属于正常水平"即可）
- 建议按优先级排序，不超过 3 条

## 分析自检
1. ✅ 平均分、及格率、优秀率三个关键指标是否都出现？
2. ✅ 是否正确率最低的 3 个知识点给出教学建议？
3. ✅ 是否指出了需要个别关注的学生（不只看整体）？
```
