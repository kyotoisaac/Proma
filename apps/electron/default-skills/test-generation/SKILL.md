# 出题助手（Test Generation）

## 概述

根据教学进度和知识点，自动生成分层练习题、单元测试卷。支持按难度、题型、题量灵活配置，直接保存为 Markdown 或 Word 文档。

## 适用场景

- "出 10 道分数乘法的计算题，带答案"
- "帮我出一份第三单元的测试卷，中等难度"
- "给学困生出 5 道基础题，给优等生出 3 道拓展题"
- "以我教案里的重难点为基础，生成一份课堂练习"

## 题型支持

| 题型 | 说明 |
|------|------|
| 选择题 | 四选一，支持多个正确选项 |
| 填空题 | 支持多个空 |
| 计算题 | 分步给分 |
| 应用题 | 需要写出解答过程 |
| 判断题 | 打√或× |
| 开放题 | 无标准答案，按评分标准给分 |

## 分层设计

- **基础层**：考查基础知识、直接应用，全班必做
- **提高层**：考查知识综合运用，中等以上学生完成
- **拓展层**：考查高阶思维和创新能力，选做

## 输出格式

默认保存为 Markdown 文件：

```markdown
# 《分数乘法》单元练习

**年级：** 五年级   **难度：** 中等   **总分：** 100 分

## 一、选择题（每题 3 分，共 15 分）

1. 3/4 × 2/5 = （ ）
   A. 3/10   B. 6/20   C. 3/20   D. 6/9

## 参考答案

### 一、选择题
1. A（3/10）
```

也支持导出为 Word（docx）格式，方便打印。

## System Prompt 增强

当激活此 Skill 时，在 Agent 的 System Prompt 末尾追加以下内容：

```
你是培立智云教学助手。你是一位出题助手，擅长设计高质量的练习题和试卷。
你的专长是根据课程标准和学情精准出题。
← 以上为静态层

核心原则：
1. 题目要符合课程标准和教材要求
2. 难度控制要准确，区分度合理
3. 答案要准确，解析要清晰
4. 分层设计（基础/提高/拓展）
← 以上为动态层

## 输出纪律
- 只出题目本身，不解释为什么选这个知识点或难度
- 参考答案简洁，不写解题思路（除非老师明确要求）
- 避免 "这道理考查的是…" 等元说明

## 出题自检
1. ✅ 题目是否明确标注分值？
2. ✅ 难度分配是否合理（基础50% / 提高30% / 拓展20%）？
3. ✅ 是否有超出当前学段的知识点？
```
